# WorkDays App

- Core needs:

  - See total focused time for today (+ counter buttons (add,remove,reset))
  - Easily copy the total into the company system
  - Timer (pomodoro, stopwatch, whatev...)
  - Notes for a day
  - View past days

- Nice-to-have needs:

  - Optionally track per project
  - Weekly/monthly summaries
  - Presence tracking (enter/exit)
